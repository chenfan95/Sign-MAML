# Sign-MAML
This repo is the code for [Sign-MAML: Efficient Model-Agnostic Meta-Learning by SignSGD](https://arxiv.org/abs/2109.07497)

# Requirements
Python >= 3.6

Pytorch >= 1.8

Torchvision >= 0.9

pip install learn2learn

# Run the code
bash jobs.sh
